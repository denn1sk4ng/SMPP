[Stock Market Price Prediction System Using LSTM]

This project is a Final Year Project developed for academic and learning purposes. It is a web-based stock market price prediction system that uses machine learning and deep learning models to predict future stock closing prices based on historical stock market data.

[PROJECT OVERVIEW]

The system allows users to upload or fetch stock market datasets, train prediction models, and generate future stock price predictions. The main model used in this project is Long Short-Term Memory (LSTM), with Linear Regression and Moving Average used as comparison models.

[MAIN FEATURES]

- User registration and login
- Dataset upload and automatic dataset fetching
- Dataset preview before training
- LSTM model training
- Linear Regression and Moving Average comparison
- RMSE-based model evaluation
- Future stock price prediction
- Market trend interpretation
- Prediction result export

[TECHNOLOGIES USED]

- Laravel
- Python
- MySQL
- HTML, CSS, JavaScript
- LSTM
- Linear Regression
- Moving Average
